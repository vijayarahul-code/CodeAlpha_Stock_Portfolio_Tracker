# Stock Portfolio Tracker

This is a simple Stock Portfolio Tracker developed using Python as part of my CodeAlpha Python Programming Internship.

## Features

- User can enter stock names and quantity
- Calculates the total investment value
- Displays individual stock value
- Saves the final result in a text file

## Technologies Used

- Python
- Dictionary
- Loops
- Conditional Statements
- File Handling

## How to Run

1. Open the project in VS Code
2. Run the stock_tracker.py file
3. Enter stock details and check the investment result
